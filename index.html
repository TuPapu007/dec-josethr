Que es NPM I NPX ? que et permeten fer

NPM es el sistema de paquetes por defecto de Node.js Un repositorio grande utilizado para la publicacion de los projectos Node.js

NPX es una herramienta CLI cuyo propósito es facilitar la instalación y la gestión de las dependencias alojadas en el registro npm.

Perque serveix el paquet express de NodeJS?

Express es uno framework web de los mas populares de Node y en su libreria tiene un gran numero de otros frameworks

  • Escritura de manejadores de peticiones con diferentes verbos HTTP en diferentes caminos URL
  • Integración con motores de renderización de "vistas" para generar respuestas mediante la introducción de datos en plantillas.
  • Establecer ajustes de aplicaciones web como qué puerto usar para conectar, y la localización de las plantillas que se utilizan para renderizar la respuesta.

Que es blocking i non-blocking en NodeJS?

El bloqueo es cuando la ejecución de JavaScript adicional al proceso Node.js debe esperar hasta que finalice una operación que no sea JavaScript

Una llamada sin bloqueo a JavaScript proporciona una función de devolución de llamada que se debe llamar cuando se haya completado la operación.
